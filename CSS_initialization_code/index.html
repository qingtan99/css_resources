看完上面的重置样式，是不是觉得它们都是大同小异，其实到了现在，css的默认样式在各浏览器中的差别已经越来越少，可以说是微乎其微，需要重置的样式，我们应该根据实际情况添加，不要照搬“大厂”的代码，否则多出很多冗余的代码，看着不舒服，也影响性能。

我的做法
*:before, *:after {
    box-sizing: border-box;
}

body,h1,h2,h3,h4,h5,h6,p,ul,ol,form {
    margin:0;
}

ul,ol {
    list-style-type:none;
}

a {
    color: #fff;
    text-decoration: none;
}

a img {
    border:0;
}

img{
    border:0;
    vertical-align: middle;
}
